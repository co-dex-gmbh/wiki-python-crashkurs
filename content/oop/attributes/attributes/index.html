    Durchschnittsnote berechnen

    class Buchhaltung:
    pass


booking = Buchhaltung()
booking.food_spending = 100
booking.car_spending = 230
booking.february_income = 200

# Lösung mit Schleife
summe = 0
for name, value in booking.__dict__.items():
    if 'spending' in name:
        summe += value

print(summe)

# Lösung mit Listcomprehension
print(sum(value for name, value in booking.__dict__.items() if 'spending' in name))

    Die erste Lösung verwendet eine Schleife, um durch alle Attribute zu iterieren und die Summe zu berechnen. Die zweite Lösung nutzt eine Listcomprehension, um dasselbe Ergebnis in einer kompakteren Form zu erzielen.

    class Person:
    pass

anna = Person()
anna.name = "Anna Lena Zitrova"

karl = Person()
karl.name = "Karl Gustav"


def getInitials(person):
    return "".join(word[0] for word in person.name.split())

print(getInitials(anna)) # 'ALZ'
print(getInitials(karl)) # 'KG'

x = Roboter()

if not hasattr(x, "speed"):

print(f"Roboter fährt {x.speed} km/h")

if hasattr(x, "marke"):

    Neben hasattr gibt es die vorimplementierten Funktionen getattr und setattr. Finde heraus, was diese Funktionen tun und ändere den obigen Code so, dass diese Funktionen verwendet werden.

    Was macht der dritte Parameter von getattr?

    Lösung
    Lösungsvideo

    hasattr prüft, ob die Instanz ein Attribut oder Klassenattribut besitzt.

    setattr setzt ein Attribut.

    getattr gibt einem das Attribut mit dem gegebenen Namen zurück. Der dritte Parameter ist optional und wird zurückgegeben, wenn kein Attribut des angegebenen Namens gefunden wurde.

    print(f"Roboter hat die Marke {getattr(x, 'marke')}")
